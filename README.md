## Installation

[Click here](https://davidkuyk.github.io/25plus5timer/) to go to the site. Or, if you want to clone this repository, open your terminal and type:

```sh
git clone https://github.com/davidkuyk/25plus5timer.git
cd 25plus5timer
```
